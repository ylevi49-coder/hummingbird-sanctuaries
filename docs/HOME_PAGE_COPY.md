HOME PAGE COPY

This document defines the official homepage copywriting direction for Hummingbird Sanctuaries.

The homepage copy must feel:

emotional
cinematic
elegant
human
premium
trustworthy
inspiring

The copy should never feel:

corporate
generic
overly salesy
templated nonprofit language
startup-like

The writing style should combine:

emotional storytelling
luxury calmness
humanitarian dignity
premium simplicity
