HOME PAGE WIREFRAME

The homepage should feel like a cinematic emotional journey.

Each section must flow naturally into the next.

The emotional pacing should move from:

emotional curiosity
→ trust
→ understanding
→ emotional connection
→ credibility
→ inspiration
→ donation action

The homepage must never feel crowded or overly aggressive.

The user experience should feel calm, elegant, immersive, and premium.

GLOBAL FLOW PRINCIPLES
Flow Style
cinematic pacing
strong spacing
emotional transitions
visual breathing room
alternating light/dark sections
immersive storytelling
