COMPONENT SYSTEM

This document defines the reusable design and layout components for the Hummingbird Sanctuaries website.

The website must always be built using reusable systems and consistent visual rules.

Never design random one-off sections without following this component system.

GLOBAL COMPONENT RULES
All components must feel:
cinematic
premium
calm
emotional
spacious
trustworthy
elegant
Components must maintain:
consistent spacing
consistent typography
consistent button styles
consistent shadows
consistent border radius
consistent animation language
GLOBAL SPACING RULES
Section Padding

100px top and bottom spacing.

Container Width

Maximum width: 1200px.

Border Radius

Use rounded corners between 18px–28px.
